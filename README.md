# 🖥️ Process Scheduling Visualizer

A Python-based project to simulate and visualize CPU scheduling algorithms used in Operating Systems.

---

## 📌 Algorithms Implemented
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Round Robin (RR)

---

## ✨ Features
- Gantt Chart Visualization 📊
- Waiting Time Calculation ⏳
- Turnaround Time Calculation 🔄
- User Input for algorithm selection

---

## 🛠️ Technologies Used
- Python
- Matplotlib

---

## ▶️ How to Run

1. Install required library:
   pip install matplotlib

2. Run the program:
   python main.py

---

## 📊 Sample Output
- Displays Gantt Chart for selected scheduling algorithm
- Shows Waiting Time and Turnaround Time for each process

---

## 🎯 Objective
To understand and demonstrate CPU scheduling techniques through visualization.
